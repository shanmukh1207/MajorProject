## 📌 Express Router (Simple Explanation)

**Express Router** is used to organize your routes by dividing them into separate files instead of writing everything in one `app.js`.

It works like a **mini Express app** where you define routes (GET, POST, PUT, DELETE) and then connect them to the main application using `app.use()`.

This helps in keeping your code:

* clean
* easy to manage
* scalable for large projects

👉 In simple words:
**Express Router helps you group related routes (like listings, reviews, users) into different files and keeps your project well-structured.**

---

## 📌 Key Point (One Line for Revision)

**Express Router = Tool to organize routes into separate modules for better structure and scalability.**
